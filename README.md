## Git Commands:

Right click inside folder, more options, Git Bash

```

git commit -m "Message"  # Make a checkpoint/snapshot
git push                 # Upload to github

git checkout <commit>    # change the repo (all tracked filed) to that snapshot
git checkout main        # change the repo to latest version
```
